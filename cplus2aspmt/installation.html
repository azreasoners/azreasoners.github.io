Installation

Installing dependencies

Ensure that your system has f2lp, gringo and dReal installed.
Install f2lp from here.
Install gringo from here.
Install dReal from here.

Downloading Binaries

Ensure that the following binaries are installed as system files(i.e in /usr/local/bin/):

  • C+ to ASPMT Translator: link
  • ASPMT Compiler: link
  • ASPMT2SMT system script: link
  • Cplus2ASPMT system script: link
NOTE: These binaries run only on 64 bit Linux Machines. Sources to build the translator an compiler can be found in the following links.
  • C+ to ASPMT Translator: Github Link
  • ASPMT Compiler: Github Link


Running the system

The system can be run by simply running the Cplus2ASPMT system script and providing the necessary query values. Further documentation and tutorials can be found at Tutorials. Examples can be found at Examples.
