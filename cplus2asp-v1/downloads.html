Downloads

The following files for the Cplus2ASP system are available for download:

  • cplus2asp_system.tar.gz - A complete archive of source code and statically-compiled Linux binaries for the Cplus2ASP system. Includes the cplus2asp, cplus2asp.bin, as2transition, and f2lp programs.
  • sample_domains.zip - A collection of sample domain files for use with Cplus2ASP, useful for reference and as examples of the syntax and features supported by Cplus2ASP.
