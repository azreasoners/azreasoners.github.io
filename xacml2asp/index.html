XACML2ASP

Automated XACML 3.0 Policy Reasoning based on Answer Set Programming

XACML2ASP is an automated XACML 3.0 policy reasoning tool based on Answer Set Programming. It consists of two parts: a translator that turns XACML 3.0 policy description into ASP facts, and an ASP program that simulates the Policy Decision Point(PDP). The translator and the PDP simulator together form an ASP reasoning engine. Given the policy description as ASP fact, an ASP solver can serve as a PDP to generate the decision for any access request. Any policy reasoning task is reduced to verifying whether the input ASP program as a whole(policy description and the PDP simulator) has certain properties, which can be achieved by writing and running additional queries.

In our implementation, the PDP simulation program is written as a set of first-order formulas. F2lp is used to turn arbitrary first-order formulas into normal and disjunctive logic programs, which can be parsed and solved using Clingo.

Click here to download the translator.

To use the translator, create a folder called "policies" in the same directory, and put the xacml policy files in this folder. Execute the .jar file directly. The ASP fact will be generated in a text file called "out.txt".

Click here to download the PDP simulation program.

Make sure F2lp and Clingo are installed. Run F2lp on PDP.lp as well as other asp files(policy facts, queries, etc.), and feed the output to Clingo.

You can also download one single zip file containing the translator, the pdp simulation program, and the examples.
