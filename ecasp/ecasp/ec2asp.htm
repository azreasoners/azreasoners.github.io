Event Calculus Answer Set Programming

Examples

The following files are logic program encodings of certain domain descriptions. These have been directly downloaded from Eric Mueller's page:

http://decreasoner.sourceforge.net/csr/ecas/index.html

1. EC/DEC Sleep_Mueller.lp

2. EC/DEC KitchenSink_Mueller.lp

3. EC/DEC PickUp_Mueller.lp

 

The .e files given below are event calculus descriptions corresponding to the above files and the .lp files are the logic programs obtained by applying EC2ASP.

1. EC/DEC Sleep.e, EC/DEC Sleep.lp

2. EC/DEC KitchenSink.e, EC/DEC KitchenSink.lp

3. EC/DEC PickUp.e, EC/DEC PickUp.lp

 

Finally, the files given below are the logic program encodings of the EC and DEC axioms by Mueller (identified by ..._Mueller.lp) and the encodings obtained by applying EC2ASP.

EC_Mueller.lp, DEC_Mueller.lp, EC.lp, DEC.lp.

To run the encodings by Erik Mueller, refer

http://decreasoner.sourceforge.net/csr/ecas/index.html.

To run the encodings obtained using EC2ASP, use the following:

lparse -c maxstep=N domain_description.lp EC/DEC.lp.

Replace N in the above line by the maximum time point.
