31  * v3.1 - Secondary restructure of parsing mechanism.

35  - Other compatibility changes for cplus2asp version 3
36  * v3.0 - Major program restructure.
37  * - Now exposes an API to allow for parsing answer sets at the program level.
38  * - Rewrite of the parser driver to increase stability and allow for complete oClingo compatibility.
39  * v2.2 - Added nicer handling of timeless predicates of the forms p, eql(p,v), and eq(p,v) to allow for non-transitional answer sets to be displayed nicely.
40  * v2.1 - Added the '--none-alias' command line option to allow for aliased values of none.
41  * v2.0 - Extended to allow for abnormality constants (staticAbnormality / dynamicAbnormality).
42  * - Now uses four different top-level predicates to classify constant types.
43  * - h/2 - fluents
44  * - occ/2 - actions
45  * - ab_h/2 - static abnormalities
46  * - ab_occ/2 - dynamic abnormalities
47  * - No longer requires fluent/1, actions/1, and rigid/1 predicates within the answer set.
48  * v1.8 - Now exits with code 100 if no solutions are found, 101 if solutions are found, 1 if an error occurs, or 0 if there's nothing to do.

50  * v1.7.2 - Optimized solution display routine, fixed extra space in solutions with no "timeless" predicates output, fixed bug where a solution with only actions wouldn't display properly.
51  * v1.7.1 - Fixed a bug where bad time stamps could appear with "timeless" predicates.
52  * v1.7 - Overhauled predicate detection, enabled support for new rigid and h-style predicate constructs, added option to show negated predicates.
53  * v1.6.1 - Fixed bug where a time step with all false predicates would not be displayed.
54  * v1.6 - Changed name to as2transition, added auto-hiding of "contribution" predicates (and option to show them anyway).
55  * v1.5 - Added capability to read output from cmodels and smodels in addition to clasp (& family), along with rudimentary ability to try to read an unknown/unidentified solver's output. Reduced memory usage and optimized handling of multiple solutions.
