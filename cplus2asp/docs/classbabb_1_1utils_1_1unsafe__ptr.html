Detailed Description

template<typename T>
class babb::utils::unsafe_ptr< T >

A lighter version of ref_ptr that can be used in unions (no constructor/destructor) This pointer assumes that a value is assigned when useage begins and NULL is assigned before usage ends.

TThe type which it's referencing. Should ONLY be descendant of Reference.
