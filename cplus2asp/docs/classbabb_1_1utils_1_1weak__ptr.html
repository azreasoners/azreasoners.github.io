Detailed Description

template<typename T>
class babb::utils::weak_ptr< T >

Weak pointer for referenced counted memory locations.

Automatically set to null when the memory location is deallocated.

Parameters
TThe type which it's referencing. Should ONLY be descendant of Reference.
