Examples

We used several examples as benchmarks of performance and analysis of the C+ to ASP translation framework. Listed below are a number of the sample domains, along with downloadable files for Cplus2ASP-compatible versions of each.

A more complete set of examples can be downloaded at the bottom of the page and is also packaged with the Cplus2ASP source code under the 'examples' sub-directory. (Please see the Obtaining Cplus2ASP page for details on obtaining the source files).

Examples from Nonmonotonic Causal Theories

The simple examples from the paper "Nonmonotonic Causal Theories" serve as good baseline domains to verify the functionality of various aspects of the CCalc language. Each covers different properties of causal logic, including associated syntax and constructs.


Domain Files
Going to Work going.cp
Lifting the Table lifting.cp
Monkey & Bananas monkey.cp
Pendulum pendulum.cp
Publishing Papers publishing.cp
Shooting Turkeys shooting.cp


Zoo World and Traffic World

Note: The Zoo domain does not compile with Cplus2ASP v3/3.1 since not all features of the language are not currently implemented.

The Zoo World and Traffic World are two non-trivial domains described by the paper "Representing the Zoo World and the Traffic World in the Language of the Causal Calculator" which demonstrate the expressiveness of C+. From The paper:

"The Zoo World consists of several cages and the exterior, gates between them, and animals of several species, including humans. Actions in this domain include moving within and between cages, opening and closing gates, and mounting and riding animals. The Traf?c World includes vehicles moving continuously between road crossings subject to a number of restrictions, such as speed limits and keeping a ?xed safety distance away from other vehicles on the road."


Domain Files
Traffic World traffic.tar.gz
Zoo World zoo.tar.gz


Additive Examples

Additive Fluents are described by the paper Describing Additive Fluents in Action Language C+. and provide construct which allows the value in the next state to be the sum of all the affects in a transition. Below we provide a number of programs demonstrating the usage of additive fluents.


Domain Files
Buying buying.cp, buying.query
Missionaries and Cannibals mcp.cp, mcp.query
Blocks World w/ Cost bw-cost.cp, bw-cost.query
Spacecraft Integer spacecraft.cp, spacecraft.query
