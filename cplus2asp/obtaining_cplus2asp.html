Obtaining Cplus2ASP

CSE 591: Foreword

If you're enrolled in a class with Dr. Joohyung Lee you probably have access to the Reasoning research server. The Reasoning research server has many tools Artificial Intelligence tools pre-installed and ready to run, including Cplus2ASP v3.0.

If you've been granted access to the Reasoning research server and do not wish to install the Cplus2ASP system, you can log in to the server over SSH by executing the following command (replace asurite_id with your Asurite ID you use to log in to your My ASU page). 

	ssh asurite_id@reasoning-server.fulton.ad.asu.edu

When prompted, type in your Asurite ID password that you use to login to your My ASU page.

If you're using a Windows system, you can still access the Reasoning server via a third-party SSH application, such as PuTTY. If you wish to use PuTTY, you should enter 'reasoning-server.fulton.ad.asu.edu' for the host name, press the 'Open' button, and use your Asurite credentials when prompted.

Obtaining Cplus2ASP's distributable files

The Cplus2ASP project is currently hosted publicly at GitHub.com and can be obtained in several ways. If you wish to obtain a one-time snapshot of the Cplus2ASP source project, you can either download the .zip file from GitHub using your web browser, or via Linux command line using the following commands:

Cplus2asp v3.1

	
	wget https://github.com/samidhtalsania/cplus2asp/archive/master.zip
	unzip master.zip

Cplus2asp v3

	
	wget https://github.com/babb517/cplus2asp/archive/master.zip
	unzip master.zip

If you download the ZIP file instead of using Git you may need to download and install as2transition, which can be found here

However, if you want to keep up on the latest versions, it's recommended that you clone the Git repository instead, which will allow you to easily update source code and track changes later. One way to accomplish this is using the following Linux command line command, assuming your system has Git installed:

Cplus2asp v3.1

	git clone --recursive git://github.com/samidhtalsania/cplus2asp.git

Cplus2asp v3

	git clone --recursive git://github.com/babb517/cplus2asp.git

If your system doesn't have Git, you will have to install it. If your system is a Debian/Ubuntu/Mint (or any other Debian based distribution) then this can be accomplished with the command: 

	sudo apt-get install git
