lpod2asprin

An implementation to compute LPOD programs using existing asprin solver

Logic Programs with Ordered Disjunction (LPOD) is an extension of standard answer set programs to handle preference using the high-level construct of ordered disjunction.

asprin is a general, flexible, and extensible framework that provides low-level constructs for modeling preference in answer set programming.

We present a new and more efficient implementation of LPOD, which compiles LPOD into the input language of asprin.
