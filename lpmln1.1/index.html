LPMLN 1.1

An implementation to compute LPMLN programs using existing ASP solvers

LPMLN 1.1 is an updated version of previous LPMLN System. It not only optimized the code structure of previous system, but also added many new features we recently developed, including LPMLN weight learning, and sampling based inference.
