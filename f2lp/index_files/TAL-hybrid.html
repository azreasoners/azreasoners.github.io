Hybrid Temporal Action Logic Theories - Example The following package contains the files needed to run the example discussed in the paper "Reformulating Temporal Action Logics in Answer Set Programming" by Joohyung Lee and Ravi Palla published in AAAI 2012.

TAL hybrid theories package.

Contents of the package:

facts-prog.lp: An ASP program that generates the necessary instances by using the rdf-plugin of DLVHEX. In our experiments, we used version 1.7.4 of DLVHEX and version 1.7.0 of the rdf-plugin.

example-1-query-1.f2lp: The example in the AAAI paper with the first query enabled. This query is to check if Aspirin can be administered to the patient.

example-1-query-2.f2lp: The example in the AAAI paper with the second query enabled. This query is to check if Acetaminophen can be administered to the patient.

usecase1-durative.f2lp: This is the TAL description of the usecase. It contains some domain knowledge dealing with administering of drugs.

f2lp-v1.2-beta: This is version 1.2(beta) of F2LP.

f2lp-v1.2-beta.c: This is the source file of the above binary.

f2lpdlv.py: The python file that turns the output of F2LP into the syntax of DLV. This is a preliminary version that may not work for all cases.

Steps to run the example:

1. Start python
2. 'import f2lpdlv'

To run example-1-query-1.f2lp, use:
f2lpdlv.run('f2lp-v1.2-beta facts-prog.lp example-1-query-1.f2lp usecase1-durative.f2lp','dlvhex --filter=holdsAt')

The program facts-prog.lp uses the National Drug File (NDF) ontology in OWL format (NDF-RT.owl). The file path of the NDF ontology needs to be replaced with the actual path in the first rule of facts-prog.lp. The following relationship needs to be added to the drug Aspirin (URI: "http://evs.nci.nih.gov/ftp1/NDF-RT/NDF-RT.owl#N0000145918") in the NDF ontology before running the above command:

<rdfs:subClassOf>
  <owl:Restriction>
   <owl:onProperty rdf:resource="#CI_with"/>
   <owl:someValuesFrom rdf:resource="#N0000001482"/>
  </owl:Restriction>
</rdfs:subClassOf>
This expresses the knowledge that Gastrointestinal Bleeding is a contraindication for administering Aspirin.
